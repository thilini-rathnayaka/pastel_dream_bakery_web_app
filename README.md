# 🍰 Pastel Dreams Bakery Web Application

## 📌 Project Overview
Pastel Dreams Bakery is an interactive and creative web application designed to simulate a real digital bakery experience. The system allows users to explore cake designs, filter categories, and place custom cake orders in an engaging and visually appealing environment.

This project focuses on combining creativity with functionality using modern web technologies.

---

## 🎨 Theme
- Pastel aesthetic colors  
- Elegant black & gold accents  
- Dreamy animations  
- Magical interactive experience  

---

## 🎯 Objectives
- Provide an attractive interface for cake browsing  
- Allow users to filter and explore cake categories  
- Enable custom cake ordering  
- Deliver a smooth and interactive user experience  

---

## 🚀 Features

### 🌐 Core Features
- Home Page  
- Cake Menu Page  
- Order/Contact Page  
- Responsive Design (Mobile, Tablet, Desktop)
